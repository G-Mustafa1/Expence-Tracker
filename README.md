# 💸 Expense Tracker

A simple full-stack Expense Tracker application that helps users manage their daily expenses. Users can add, edit, and delete expenses, and view their spending in an organized way.

# 📌 Features

User Signup & Login

Add, Edit, Delete Expenses

View all expenses in one place

Search and filter expenses

Dashboard with basic stats

Protected routes (only logged-in users can access data)
